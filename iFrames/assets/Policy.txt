<span class="title">2. Information We Collect</span><br>

Spud collects and stores data that users submit to configure various features, including:
<ul>
    <li>Server welcome messages</li>
    <li>Verification system settings</li>
    <li>Other configurations made through the bot</li>
</ul>

<span class="subtitle">2.2 Member Data</span><br>
Spud collects basic member data from Discord to provide its services. This includes:
<ul>
    <li>Username</li>
    <li>User ID</li>
    <li>Avatar</li>
</ul>
We only store the User ID in our database to support the leveling system functionality, other data is not stored or kept.
<br><br>
<span class="subtitle">2.3 Dashboard Data</span><br>
When users log in to Spud's dashboard using their Discord account, we collect and store necessary data in a temporary session. This session data is deleted after a server restart or after a certain period.

<span class="subtitle">3.2 Member Data</span><br>

<span class="subtitle">3.3 Dashboard Data</span><br>
