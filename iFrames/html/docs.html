Member issues

Interaction failed

If the bot displays "This interaction failed" you have run into an issue regarding the button collector. This is most likely caused by waiting too long or if the bot has restarted. If this occurs, please retake the verification.

Verified role not granted

Finished the verification/tutorial, but didn't get a verified role causing you to not be able to access the rest of the server? Please contact a server staff member; this issue is most likely caused by the role ID being invalid.

Setup tutorial

1 - Getting started

Before we start the tutorial, make sure your server has a channel where the verify message will be displayed and make sure you have set up a role to give to new members who finish the verification/tutorial.

2 - Selecting a channel

Run the /database set command and choose Verify channel at the Type field. After this, select the preferred channel from the list.
Upon finishing this task, the setup menu will open.

3 - Setting a verified role

Next step is setting a role to grant users who complete the verification/tutorial. This can be achieved by clicking the Verify role button on the setup menu, in the menu that opens type or paste in the ID of the preferred role.

4 - Adding a page

Next step is adding your own page to the verification/tutorial system! Click the Add button on the setup menu and fill in the fields on the menu that opens.

The page cooldown will be the timer before the new member can go to the next page. e.g. 10000 = 10 seconds

5 - Editing a page

Made a mistake, or don't like one of the pages, but don't want to change the page order? Edit a page by clicking the Edit button on the setup menu! You can select the page you want to edit using the dropdown menu.

6 - Removing a page

Not happy with one of the pages? Remove the unwanted page(s) by clicking the Remove button, same as edit this will display a dropdown menu where you can select 1 or more pages to remove.

7 - Publishing

Finished configuring? Pressing the Publish button will make the verify system public to new members.
